# Primitive checklist

Use this checklist before considering a primitive complete.

- [ ] Lives in `src/primitives`
- [ ] Does not depend on legacy `S*` components
- [ ] Uses token-only styling
- [ ] Avoids raw Tailwind color utilities
- [ ] Avoids arbitrary spacing, radius, and motion values
- [ ] Uses consistent prop naming
- [ ] Has documented variants and sizes
- [ ] Has relevant interactive states
- [ ] Has visible focus treatment if interactive
- [ ] Meets accessibility expectations
- [ ] Has a `*.spec.md` file
- [ ] Is exported intentionally
- [ ] Is clearly a primitive rather than a pattern or app component

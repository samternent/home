# `@ternent/pixpax-concord`

PixPax v2 Concord plugin, commands, and replay projection.

This package keeps PixPax-specific meaning above generic Concord:

- `pixbook.claim-pack`
- `pixbook.record-pack-opened`
- `pixbook.set-display-profile`

Ownership derives from claims only. Opened state is UX-only.

# `@ternent/pixpax-core`

Core PixPax v2 domain rules, canonical types, deterministic derivation, and replay-facing helpers.

This package owns:

- canonical claimant normalization
- deterministic issuance material
- pack item and root hashing
- stable card instance ids
- Pixbook projection helpers

This package does not own:

- Concord runtime wiring
- issuer-side signing
- API storage or policy enforcement
